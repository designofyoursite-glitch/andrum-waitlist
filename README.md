# Andrum landing page

Static landing page for Andrum. The project is intentionally framework-free: HTML, CSS, JavaScript, and assets can be hosted on GitHub Pages or any static hosting.

## Structure

```
.
├── index.html
├── assets
│   ├── css
│   │   └── styles.css
│   ├── js
│   │   └── main.js
│   ├── images
│   │   └── team.jpg
│   ├── og-image.jpg
│   ├── og-image.svg
│   ├── ANDRUM.svg
│   └── Andrum_full.svg
├── .gitignore
└── README.md
```

## Local preview

Open `index.html` in a browser, or run a static server from this folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
